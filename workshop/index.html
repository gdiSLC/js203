Fun practice APIs

These APIs all have really great documentation and are easy to work with for a newcomer to data APIs. Some have options for user data and authentication as well.

  • Big Huge Thesaurus: thesaurus
  • YouTube: videos, optional user authentication and data
  • SoundCloud: music, optional user authentication and data
  • Sunlight Foundation: government data
  • Forecast.io: weather
