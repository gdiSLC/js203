img as API

A type of simple HTML API is the img tag. It looks like this:

<img src="http://otherdomain.com/service?size=100&color=red">

Broken down:

  • http://otherdomain.com/service is the site we are sending the query to
  • ?size=100&color=red are the query parameters. We have two parameters: size (value: 100) and color (value: red).
